  title = {{Tools for Self-Organizing Applications Engineering}},
  year = {2003},
  abstract = {Nowadays, the applications to be realized are more often distributed, complex and open, e.g., applications on the Internet: information search, brokerage, e-commerce, e-business... Therefore, designers cannot implement a global control and list all situations such systems have to be faced with. ADELFE1 methodology was proposed to develop this kind of software. It is based on the AMAS theory (Adaptive Multi-Agent Systems) and the emergence concept. This theory gives local agent design criteria so as to enable the emergence of an organization within the system and thus, of the global function of the system. This paper focuses on three tools of the methodology associated with the process and the UML/AUML notations. The first tool is based on the commercial software OpenTool, enriched to take into account adaptive multi-agent system development. The second tool is a support decision tool to help designers to decide if the AMAS theory is relevant for the current system to design. The last tool is an interactive tool which supports the process and helps designers to follow the process and to execute associated tasks. The use of each tool is illustrated by ETTO (Emergent TimeTabling Organization) application.},
